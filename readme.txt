== Description ==

Display a box with the biography and social media presence of one or more author(s) of the post.

== Features ==
 * Displays gravatar
 * Option of choosing the colors
 * Choice of displaying contributors on home page too.

= Credits =

* Initial idea by Menaka S. inspired by [rtcamp assignment 2b](https://careers.rtcamp.com/wordpress-engineer/assignment/).

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Settings -> Rapo MultiAuthors and fill the options;
* Fill the **Biographical Info** and **Contact Info** in Users -> Your Profile.
